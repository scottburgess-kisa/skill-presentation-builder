# Presentation Templates

This directory contains template specifications for different presentation styles.

## Available Templates

### [defra.md](defra.md)
UK Government DEFRA (Department for Environment, Food and Rural Affairs) branding template.

**Features:**
- DEFRA green colour scheme
- Government branding requirements
- Equal Experts supplier designation
- Official footer markings

## Adding New Templates

To add a new presentation style:

1. Create `templates/your-style.md` following the template format
2. Include all required sections: Identity, Assets, Colours, Fonts, Slide Types
3. Update the skill to reference the new template
4. Add corresponding assets in `assets/your-style/`

## Template Structure

Each template should define:
- **Identity:** Slide dimensions and basic properties
- **Assets:** Logo files, backgrounds, and brand elements
- **Colours:** Brand colour palette with hex codes and usage
- **Fonts:** Typography specifications for different roles
- **Slide Types:** Layout specifications with precise dimensions
- **Placeholder Box Specification:** Styling for image placeholders
